# Audio Reference Files

This directory contains large audio reference files used for testing.

## Required Files

| File | Size | Description |
|------|------|-------------|
| `record_out.wav` | ~17MB | Test recording audio |

## Download

Check the project documentation or re-record the audio using the recording scripts in the project.

If this file was previously tracked, it can be restored from an earlier git commit:

```bash
git show HEAD:audio_refs/record_out.wav > audio_refs/record_out.wav
```
