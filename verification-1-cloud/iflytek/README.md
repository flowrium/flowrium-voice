# 讯飞开放平台

## 定位

讯飞在中文语音技术领域长期有较强存在感。虽然不是传统大而全云厂商，但在语音识别、语音合成和教育场景里经常是重点候选。

## 厂商类型

语音技术平台。

## 与本验证相关的产品

- `语音听写`：实时或短语音识别
- `录音文件识别`：离线音频转写
- `语音转写`：长音频、会议类或连续语音场景
- `热词 / 个性化能力`：专有名词增强
- `开放平台控制台`：应用管理、鉴权、配额查看

## 接入方式

- 控制台开通
- HTTP API
- WebSocket API
- 官方 SDK

## 优势

- 中文语音场景经验深
- 教育、普通话、连续语音类项目中常被采用
- 适合直接做 ASR 能力验证

## 风险和注意点

- 不同语音产品的接口形态可能不统一
- 鉴权和流式接入复杂度需要尽早评估
- 需要区分短语音、长语音和实时语音能力边界

## 验证关注点

- 中文连续指令识别稳定性
- 校长场景专有词识别效果
- WebSocket / HTTP 接口接入复杂度
- 免费额度、并发和限流

## 记录建议

- 重点记录中文连续说话时的断句与漏词情况
- 单独跟踪教育场景专有词和人名表现
