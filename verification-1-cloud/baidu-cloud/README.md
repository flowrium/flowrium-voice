# 百度智能云

## 定位

百度智能云在语音、视觉、NLP 等 AI 能力上有长期积累，适合作为国内语音识别候选之一。

## 厂商类型

AI 平台型云厂商。

## 与本验证相关的产品

- `语音技术 / ASR`：短语音、实时语音、长语音识别
- `对象存储 BOS`：音频文件存储
- `云服务器 BCC`：自建测试和代理服务
- `函数计算 CFC`：轻量事件处理
- `日志服务`：调用追踪和质量分析

## 接入方式

- 控制台开通
- API 调用
- 官方 SDK

## 优势

- 中文语音能力积累较早
- AI 产品线完整，适合快速验证
- 作为国内厂商对照组有价值

## 风险和注意点

- 不同语音产品的适用边界需要提前确认
- 文档清晰度和调试体验要纳入工程成本评估
- 热词和定制能力不要只看宣传，需要样本实测

## 验证关注点

- 长音频和短指令两类场景的表现差异
- 热词、自定义词汇和行业词支持
- 接口文档清晰度和 SDK 易用性
- 免费额度和计费方式

## 记录建议

- 单独比较短指令和长音频的识别差异
- 把接口易用性和调试成本写进最终结论
