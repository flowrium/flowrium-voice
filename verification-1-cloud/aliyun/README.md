# 阿里云

## 定位

阿里云是国内产品线最完整的公有云之一，适合中国大陆部署、对象存储、后端服务和 AI 能力接入。

## 厂商类型

通用云厂商。

## 与本验证相关的产品

- `智能语音交互`：语音识别、语音合成、实时与离线语音能力
- `对象存储 OSS`：音频文件暂存、归档、结果回放
- `函数计算 FC`：轻量转写任务编排
- `ECS`：自建转写网关或批处理服务
- `日志服务 SLS`：转写日志、错误追踪和性能统计

## 接入方式

- 控制台开通
- API 调用
- 官方 SDK

## 优势

- 国内接入和文档体系成熟
- 常规云资源和语音能力容易组合
- 适合做中国大陆业务的基础设施落地

## 风险和注意点

- 语音能力和通用云资源分属不同产品线，需要确认实际开通流程
- 费用结构需要单独核对，避免只看通用云价格
- 热词和领域定制能力需要实测，不适合先假设

## 验证关注点

- 普通话识别准确率
- 热词和专有名词支持
- 文件转写和实时转写接口差异
- 价格和调用限额

## 记录建议

- 记录同一批音频在实时接口和文件接口上的差异
- 单独标记学校名称、班级、日期、人数等关键词
