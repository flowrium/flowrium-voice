# 火山引擎

## 定位

火山引擎是字节系技术服务平台，在音视频、推荐、增长和 AI 场景中较活跃，语音相关能力值得纳入对比。

## 厂商类型

通用云与特色 AI 平台。

## 与本验证相关的产品

- `语音识别`：实时和录音文件转写能力
- `对象存储 TOS`：音频文件管理
- `ECS`：自建测试服务和代理层
- `日志服务 TLS`：调用日志和质量分析
- `RTC / 音视频能力`：如果后续扩展到实时互动可复用

## 接入方式

- 控制台开通
- API 调用
- 官方 SDK

## 优势

- 音视频和实时处理能力有特色
- 国内产品更新较快
- 适合与字节系 AI / 数据能力一起评估

## 风险和注意点

- 通用云资料和最佳实践沉淀可能少于头部厂商
- 语音接口的文档完整度需要单独确认
- 需要验证结果结构是否便于后处理

## 验证关注点

- 长音频转写效果
- 实时流式识别延迟
- 接口稳定性和返回结果结构
- 价格和测试门槛

## 记录建议

- 重点记录延迟、断句和结果回传格式
- 如果后续要接实时互动，单独保留接口复用性结论
