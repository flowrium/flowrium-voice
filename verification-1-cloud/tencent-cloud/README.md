# 腾讯云

## 厂商信息

腾讯云是国内通用云厂商中比较偏 `音视频 / 实时互动` 生态的一家。对本次验证，腾讯云的价值在于 `ASR 本身够完整`，同时能与 `COS / CLS / TRTC` 很自然地串起来。

## 与本验证相关的 ASR 产品

- `实时语音识别`：不限时长实时流式识别，适合字幕、对话机器人、会议实时记录。
- `录音文件识别`：异步文件转写，适合课堂、会议、质检等。
- `录音文件识别极速版`：更强调准实时返回，适合字幕生产和短视频。
- `一句话识别`：60 秒内短音频识别。
- `语音流异步识别`：流式输入但异步输出，适合特定服务端场景。
- 增值能力：`热词增强、说话人分离、情绪识别、分段、口语转书面语`。

## 支持的功能

- 实时流式识别、异步文件识别、极速文件识别、短音频识别。
- 热词增强、说话人分离、情绪识别、分段、口语转书面语等增值能力。
- `录音文件识别` 文档显示：
  - 默认接口限频 `20 次/秒`
  - 并发任务上限 `200`
  - 音频 URL 时长可到 `12 小时`
  - 音频格式支持 `wav / ogg-opus / mp3 / m4a`
- `实时语音识别` 在产品功能页标注默认 `200 路`。
- 多语种与多方言能力比较强，官方产品页强调支持 `27 种方言`，录音文件识别大模型支持 `15 个语种自动识别`。

## 优势

- 产品矩阵完整，实时、离线、极速、短音频都齐全。
- 免费额度相对友好，适合快速验证。
- 与腾讯音视频生态天然匹配，若未来有直播、会议、TRTC 场景，延展性强。
- 文档对接口限制、并发、结算方式写得较细，工程可控性较好。

## 劣势

- 产品和增值能力较多，第一次看容易被 `基础类 / 增值类 / 大模型版` 搞混。
- 后付费默认关闭，需要手工开启，否则资源包耗尽会停服。
- 大模型版和跨境版的价格会明显高于标准版，评估时要分开看。

## 对校长语音指令场景的判断

- 如果后续要和腾讯云的存储、日志、实时音视频一起做完整方案，腾讯云非常合适。
- 如果只看中文 ASR 极限表现，它通常会是稳妥但未必最惊艳的那一档，需要和讯飞、火山引擎硬比样本集。
- 如果你比较重视接口稳定性、并发管理和工程闭环，腾讯云会比较省心。

## 重点验证项

- 实时与录音接口在断句、时间戳、标点上的一致性。
- 热词增强对学校专有词、人名、课程名的收益。
- 大模型版是否显著优于标准版，以及是否值得更高价格。
- 与 COS / CLS / TRTC 组合后的整体开发体验。
