# 腾讯云

## 定位

腾讯云在国内互联网、音视频、实时通信场景很常见，适合评估语音能力与常规云资源的组合使用。

## 厂商类型

通用云厂商。

## 与本验证相关的产品

- `语音识别 ASR`：实时和录音文件识别
- `对象存储 COS`：音频上传与归档
- `CVM`：自建服务
- `SCF`：事件驱动任务处理
- `CLS`：日志检索和调用分析

## 接入方式

- 控制台开通
- API 调用
- 官方 SDK

## 优势

- 国内接入成熟
- 音视频和实时互动生态较强
- 与常规云资源整合自然

## 风险和注意点

- 需要核对 ASR 产品细分能力和限额策略
- 不同 SDK 的易用性可能有差异
- 要确认是否方便做批量验证和日志追踪

## 验证关注点

- 中文专有名词与学校场景词汇表现
- 流式接口和离线接口的结果一致性
- SDK 与鉴权实现复杂度
- 调用成本和速度

## 记录建议

- 记录实时和文件识别的结果结构差异
- 关注是否方便与 COS、CLS 串联成完整验证链路
